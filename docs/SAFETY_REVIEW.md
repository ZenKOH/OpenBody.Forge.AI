# Safety Review

This review applies to avatar animation assets, not to the user's body.

## Required checks

- Demo is labelled educational.
- Viewer includes stop, pause, replay and voice-off controls.
- Movement speed is appropriate for learning.
- Animation avoids obviously unsafe joint collapse or extreme range.
- Easier option exists for the exercise.
- Human reviewer approves educational use.

## Output labels

```text
Educational demo
Needs review
Unsafe-looking animation
Licence incomplete
Ready for app integration
```

## Boundary

OpenBody Forge AI does not diagnose, prescribe, or assess user movement. It prepares and validates avatar demonstration assets and metadata.
