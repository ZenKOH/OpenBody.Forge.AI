
## Retargeting priorities

- Preserve foot contact.
- Reduce sliding.
- Avoid extreme joint angles.
- Keep speed suitable for learning.
- Segment every clip into setup, move, control and return.

## Export contract

Every app-ready GLB should expose named clips that match the browser runtime map generated by `index.html`.
